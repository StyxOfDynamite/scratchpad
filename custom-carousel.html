Custom CSS

You can add custom css as a property of the configuration object passed into carouselWdiget(). On caveat is this will need to be on a single, to remain a valid javascript string. 

			carouselWidget('carousel-widget-810',{
			  store: 'businessexpert-co-uk',
			  primaryClr: '#f47e27',
			  neutralClr: '#f4f4f4',
			  reviewTextClr: '#494949',
			  layout:'fullWidth',
			  numReviews: 21,
			  css:'.reviewText p{font-size: 20px!important;}'
			});
